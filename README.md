 ToDo App

A simple fullstack ToDo application that allows users to manage their daily tasks efficiently.

1. Tech Stack

-- Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

-- Frontend
- React (Vite)
- Axios
- Tailwind CSS (optional)

---

2. Features
- Create, update, and delete todos
- Mark todos as completed
- RESTful API
- Simple and clean UI
